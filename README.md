# MusicHub

## Pages

- Profile
  - Prophile photo
  - Header photo
  - Bio
  - Artist tags
  - Portfolio pieces
- Dashboard
  - 'news feed'
  - upload photo
  - link to video
  - link to audio
- Explore
  - select from common tags or choose one
- Followers
  - show follow button if not following
- Following
  - shows 'follows you'
- Messages
  - individual message threads
  - share photos and links playable w/ mini player

## User Actions

1. create account/log in
2. Fill out profile
   1. photo
   2. bio
   3. artist tags
   4. genres
3. Upload content
   1. text posts
   2. photos (link)
   3. audio (link)
   4. video (link)
